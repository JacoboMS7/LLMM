Explanation of 5 SVG Tags

1. <rect> – Rectangles

Description: Used to draw rectangles and squares. Ideal for buttons, backgrounds, interfaces, or geometric elements.

  • x: horizontal position
  • y: vertical position
  • width: width
  • height: height
  • fill: fill color
<svg width="200" height="100">
    <rect x="10" y="10" width="150" height="80" fill="blue" />
</svg>

2. <circle> – Circles

Description: Draws perfect circles. Used in icons, charts, markers, or illustrations.

  • cx: X center
  • cy: Y center
  • r: radius
  • fill: fill color
<svg width="150" height="150">
    <circle cx="75" cy="75" r="50" fill="red" />
</svg>

Result:

More information

3. <line> – Lines

Description: Allows drawing straight lines. Very useful for charts, diagrams, dividers, and technical designs.

  • x1: starting X point
  • y1: starting Y point
  • x2: ending X point
  • y2: ending Y point
  • stroke: line color
<svg width="200" height="100">
    <line x1="10" y1="10" x2="190" y2="90" stroke="black" />
</svg>

4. <ellipse> – Ellipses

Description: Draws ellipses, useful for logos, charts, and visual decoration.

  • cx: X center
  • cy: Y center
  • rx: horizontal radius
  • ry: vertical radius
  • fill: fill color
<svg width="200" height="120">
    <ellipse cx="100" cy="60" rx="80" ry="40" fill="green" />
</svg>

Result:

More information

5. <path> – Advanced Paths

Description: The most versatile SVG tag. Allows drawing complex shapes using commands.

  • d: path commands
  • stroke: border
  • fill: fill
<svg width="200" height="200">
    <path d="M10 10 H 190 V 190 H 10 Z" fill="yellow" stroke="black" />
</svg>
