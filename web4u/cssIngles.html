1. color

Description: Changes the text color inside an element. It is used to highlight titles, paragraphs or text, and to define visual hierarchy.

  • Values: color names, hex, rgb, hsl.
p {
    color: red;
}

2. background-color

Description: Defines the background color of an element. It is used to highlight sections, separate visual blocks or give identity to the design.

  • Values: hex, rgb, hsl, color names.
body {
    background-color: lightblue;
}

Result:

More information

3. font-size

Description: Controls the size of the text. It is used to set visual hierarchy such as large titles and smaller secondary text.

  • Values: px, em, rem, %.
p {
    font-size: 32px;
}

Result:

More information

4. margin

Description: Defines the outer space around elements. It is used to separate boxes, paragraphs, images or any elements from each other.

  • Values: 1, 2 or 4 values (top, right, bottom, left).
p {
    margin: 50px 80px;
}

Result:

More information

5. border

Description: Creates a border around an element. It is used to highlight sections, create containers or mark visual boundaries.

  • Components: thickness, style and color.
  • Common styles: solid, dashed, dotted.
p {
    border: 2px solid black;
}
