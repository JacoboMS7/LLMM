1. Tag <a> – Links

The <a> tag is used to create links that allow the user to navigate to another page, file, or section of the document.

Main attributes:

  • href: link destination
  • target: controls whether it opens in another tab
  • title: informational text when hovering
<a href="https://www.wikipedia.org" target="_blank" title="Open Wikipedia">
Wikipedia
</a>

2. Tag <img> – Images

The <img> tag allows adding images to the page. It is an empty tag, without closing.

Main attributes:

  • src: image path
  • alt: alternative text
  • width/height: image size
<img src="paisaje.jpg" alt="Natural landscape" width="350">

3. Tags <ul>, <ol> and <li> – Lists

Unordered lists use <ul> and each item is defined with <li>.

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

4. Tag <table> – Tables

Tables allow organizing information in rows and columns.

Main elements:

  • <tr>: row
  • <th>: header
  • <td>: data cell
<table border="1">
    <tr>
        <th>Product</th>
        <th>Price</th>
    </tr>
    <tr>
        <td>Apples</td>
        <td>$2.00</td>
    </tr>
    <tr>
        <td>Bread</td>
        <td>$1.50</td>
    </tr>
</table>

5. Tag <form> – Forms

The <form> tag allows capturing user information and sending it to a server.

Main attributes:

  • action: URL where data is sent
  • method: sending method
  • id: identifies the element
  • name: name under which it will be sent
<form action="/procesar" method="POST">
    <label for="nombre">Name:</label>
    <input type="text" id="nombre" name="nombre" placeholder="Write your name">

    <label for="correo">Email:</label>
    <input type="email">
</form>
