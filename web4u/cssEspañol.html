1. color

Descripción: Cambia el color del texto dentro de un elemento. Se usa para resaltar títulos, párrafos o color, y definir jerarquía visual.

  • Valores: nombres de color, hex, rgb, hsl.
p {
    color: red;
}

2. background-color

Descripción: Define el color de fondo de un elemento. Se usa para destacar secciones, separar bloques visuales o dar identidad al diseño.

  • Valores: hex, rgb, hsl, nombres de color.
body {
    background-color: lightblue;
}

3. font-size

Descripción: Controla el tamaño del texto. Se usa para establecer jerarquía visual como títulos grandes y textos secundarios más pequeños.

  • Valores: px, em, rem, %.
p {
    font-size: 32px;
}

4. margin

Descripción: Define el espacio externo alrededor de los elementos. Se usa para separar cajas, párrafos, imágenes o cualquier elemento entre sí.

  • Valores: 1, 2 o 4 valores (arriba, derecha, abajo, izquierda).
p {
    margin: 50px 80px;
}

5. border

Descripción: Crea un borde alrededor de un elemento. Se usa para destacar secciones, crear contenedores o marcar límites visuales.

  • Componentes: grosor, estilo y color.
  • Estilos comunes: solid, dashed, dotted.
p {
    border: 2px solid black;
}
