1. Etiqueta <a> – Enlaces

La etiqueta <a> se usa para crear enlaces que permiten al usuario navegar a otra página, archivo o sección del documento.

Atributos principales:

  • href: destino del enlace
  • target: controla si se abre en otra pestaña
  • title: texto informativo al pasar el mouse
<a href="https://www.wikipedia.org" target="_blank" title="Abrir Wikipedia">
Wikipedia
</a>

2. Etiqueta <img> – Imágenes

La etiqueta <img> permite insertar imágenes en la página. Es una etiqueta vacía, sin cierre.

Atributos principales:

  • src: ruta de la imagen
  • alt: texto alternativo
  • width/height: tamaño de la imagen
<img src="paisaje.jpg" alt="Paisaje natural" width="350">

3. Etiquetas <ul>, <ol> y <li> – Listas

Las listas desordenadas usan <ul> y cada elemento se define con <li>.

4. Etiqueta <table> – Tablas

Las tablas permiten organizar información en filas y columnas.

Elementos principales:

  • <tr>: fila
  • <th>: encabezado
  • <td>: celda de datos
<table border="1">
    <tr>
        <th>Producto</th>
        <th>Precio</th>
    </tr>
    <tr>
        <td>Manzanas</td>
        <td>$2.00</td>
    </tr>
    <tr>
        <td>Pan</td>
        <td>$1.50</td>
    </tr>
</table>

5. Etiqueta <form> – Formularios

La etiqueta <form> permite capturar información del usuario y enviarla a un servidor.

Atributos principales:

  • action: URL donde se envían los datos
  • method: método de envío
  • id: Identifica el elemento
  • name: Nombre con el que se enviará
<form action="/procesar" method="POST">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" placeholder="Escribe tu nombre">

    <label for="correo">Correo electrónico:</label>
    <input type="email">
</form>
